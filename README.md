# Dog friendly places in Slovenia

As a freshly minted puppy owner, my new criteria for a potential visit to a bar/restaurant became dog-friendliness (apart from an excellent selection of beers or tasty coffee, obviously).

Inspired by [Oto Brglez](https://twitter.com/otobrglez) and his list of [co-working places](https://github.com/otobrglez/awesome-workplaces-slovenia), I started to build a list of dog-friendly places in Slovenia. Please feel free to add some more via PR or reach out to me at mia@kendu.si
